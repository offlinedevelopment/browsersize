# Browser Size
Display Browser window size in pixels. Updates automatically when screen is resized.  

Hosted on [GitHub Pages](https://offlinedevelopment.github.io/browsersize/#).

- 100% client side HTML CSS and javascript
- Works offline
- No logins
- No data collection
- No cookies
- No spyware
- No malware
- No exploits
- No adware
